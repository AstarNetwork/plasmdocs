Consensus Algorithm

Plasm chain uses NPoS which is used on the Polkadot relay chain. This consensus algorithm consists of 3 steps.

  1. A Nominator selects a validator NPoS
  2. A validator verifies transactions and make a new block BABE
  3. Finalize the block that was delivered in the network GRANDPA
